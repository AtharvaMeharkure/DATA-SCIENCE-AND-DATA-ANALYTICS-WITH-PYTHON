
# 📊 Data Science and Data Analytics with Python

Welcome to the **Data Science and Data Analytics with Python** repository! This project provides hands-on notebooks and resources for analyzing datasets, cleaning data, and extracting insights using leading Python libraries.

## 📝 Project Overview

This repository primarily features **STUDENT_ANALYSIS.ipynb**, a comprehensive notebook exploring and analyzing the Student Performance Dataset (Math course). The analysis covers the full workflow from importing data and libraries to cleaning, exploring, and visualizing student grade data.

---

## 📚 Contents

- **STUDENT_ANALYSIS.ipynb**  
  - Import and summary of Python libraries
  - Data loading and cleaning (missing values, duplicates)
  - Exploratory data analysis (EDA)  
  - Summary statistics and visualizations
  - Insights with example code and clear markdown explanations

---

## 🚀 How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/AtharvaMeharkure/DATA-SCIENCE-AND-DATA-ANALYTICS-WITH-PYTHON.git
   cd DATA-SCIENCE-AND-DATA-ANALYTICS-WITH-PYTHON
   ```

2. **Install requirements**  
   Open your terminal and ensure Python 3.x is installed.
   ```bash
   pip install numpy pandas matplotlib seaborn
   ```

3. **Run the Notebook**  
   Open `STUDENT_ANALYSIS.ipynb` in Jupyter Notebook, JupyterLab, or Google Colab.

   - If using Google Colab, simply upload the notebook and the required dataset.
   - Make sure the student dataset CSV (e.g., `student-mat.csv`) is available in your working directory or Google Drive as required by the notebook.
